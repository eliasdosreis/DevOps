Amazon Lex

1. Analogia do Dia a Dia

Lex é como um "construtor de chatbots": permite criar assistentes virtuais que entendem e respondem a comandos de voz ou texto, igual aos chatbots de atendimento online.

2. O que é (definição técnica oficial AWS)

Amazon Lex é um serviço de machine learning para criar interfaces de conversação (chatbots) usando reconhecimento de voz e processamento de linguagem natural.
