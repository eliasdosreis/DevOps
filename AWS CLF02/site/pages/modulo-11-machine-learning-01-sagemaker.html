Amazon SageMaker

1. Analogia do Dia a Dia

SageMaker é como uma "fábrica de robôs inteligentes": você fornece os materiais (dados), escolhe o modelo (robô), treina, testa e coloca para trabalhar, tudo em um só lugar.

2. O que é (definição técnica oficial AWS)

Amazon SageMaker é um serviço gerenciado que permite construir, treinar e implantar modelos de machine learning em escala, com ferramentas integradas para todo o ciclo de vida do ML.
