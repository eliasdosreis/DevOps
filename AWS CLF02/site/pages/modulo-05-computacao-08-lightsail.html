AWS Lightsail — VPS simplificado

1. Analogia do Dia a Dia

Lightsail é como um kit pronto para montar: você recebe tudo o que precisa para colocar um site ou app no ar, sem se preocupar com detalhes técnicos avançados.

2. O que é (definição técnica oficial AWS)

AWS Lightsail oferece servidores virtuais (VPS), bancos de dados, armazenamento e rede em pacotes simples, com preço fixo e interface amigável.

4. Questões de Fixação

  1. O que é o AWS Lightsail?

    2. A) Serviço de banco de dados B) VPS simplificado com preço fixo (correta) C) Serviço de armazenamento D) Load balancer

Explicação:

  • B) Correta: Lightsail é VPS simples.
  • A), C), D) não são Lightsail.

Armadilha: confundir Lightsail com EC2.

  1. Quando NÃO usar Lightsail?

    2. A) Para workloads complexos (correta) B) Para pequenos sites C) Para iniciantes D) Para preço previsível

Explicação:

  • A) Correta: Lightsail não é para workloads complexos.
  • B), C), D) são casos de uso de Lightsail.

Armadilha: achar que Lightsail substitui EC2 em todos os casos.

5. Conceito Senior

  • Lightsail facilita prototipagem e MVPs.
  • Não oferece todos os recursos avançados do EC2.

6. Resumo para Revisão

  • Lightsail = VPS simples | preço fixo | interface amigável | ideal para iniciantes | não serve para workloads complexos
