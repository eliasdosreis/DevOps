AWS CLF-C02 - Trilha de estudo para ficar forte na prova

Este material foi organizado para estudar em dois niveis ao mesmo tempo:

  • Explicacao simples: como se fosse para uma crianca de 8 anos.
  • Visao senior: como um profissional experiente pensa na escolha do servico.

Peso oficial da prova

DominioTemaPeso
1Conceitos de nuvem24%
2Seguranca e conformidade30%
3Tecnologia e servicos de nuvem34%
4Cobranca, precos e suporte12%

Prioridade pratica:

  1. Domine seguranca: responsabilidade compartilhada, IAM, MFA, least privilege, criptografia, logs, compliance.
  2. Domine escolha de servico: EC2 vs Lambda vs Fargate, S3 vs EBS vs EFS, RDS vs DynamoDB, CloudTrail vs CloudWatch vs Config.
  3. Domine custo: On-Demand, Reserved, Spot, Savings Plans, Budgets, Cost Explorer, Pricing Calculator.
  4. Treine pegadinhas: a prova quase sempre descreve um cenario e pergunta qual servico resolve melhor.

Como estudar

  1. Leia cada modulo normal.
  2. Depois leia modulo-00-guia-da-prova\01-mapa-oficial-e-lacunas.md.
  3. Use modulo-14-revisao-senior-crianca\01-comparativos-que-caem.md para revisar rapido.
  4. Use modulo-14-revisao-senior-crianca\02-pegadinhas-de-prova.md antes dos simulados.
  5. Faça modulo-13-simulado-final\03-simulado-senior-comentado.md marcando as erradas.

Regra de ouro

Na prova CLF-C02, voce nao precisa configurar servico. Voce precisa reconhecer:

  • O que o servico faz.
  • Quando usar.
  • Quando nao usar.
  • Qual opcao e mais gerenciada, mais barata, mais segura ou mais simples.
