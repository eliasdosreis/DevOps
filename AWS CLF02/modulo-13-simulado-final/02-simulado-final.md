# Simulado Final AWS CLF-C02

## 1. Analogia do Dia a Dia
O simulado é como um treino antes da competição: permite testar seus conhecimentos, identificar pontos fracos e ajustar a preparação antes da prova real.

## 2. O que é (definição técnica oficial AWS)
Simulado final é um conjunto de questões que cobre todos os domínios do exame, simulando o formato, o tempo e o nível de dificuldade da prova oficial.

## 4. Questões de Fixação
1. Por que fazer um simulado final?
A) Para testar conhecimentos e revisar (correta)
B) Para criar backups
C) Para monitorar recursos
D) Para criar VPCs

Explicação:
- A) Correta: simulado é para revisão.
- B), C), D) não são funções do simulado.

Armadilha: achar que simulado não é necessário se já estudou tudo.

2. Como usar o simulado para melhorar?
A) Identificar e revisar pontos fracos (correta)
B) Apenas acertar todas as questões
C) Ignorar questões erradas
D) Revisar só o que já sabe

Explicação:
- A) Correta: revisar erros é fundamental.
- B), C), D) não são estratégias eficazes.

Armadilha: achar que só acertar basta.

## 5. Conceito Senior
- Simulados devem ser feitos em ambiente controlado, simulando o tempo real da prova.
- Revisar explicações das questões erradas é essencial.
