# Amazon Polly

## 1. Analogia do Dia a Dia
Polly é como um "narrador digital": transforma textos em fala natural, igual aos assistentes virtuais que leem mensagens para você.

## 2. O que é (definição técnica oficial AWS)
Amazon Polly é um serviço de conversão de texto em fala (Text-to-Speech, TTS) que utiliza machine learning para gerar vozes naturais em vários idiomas.
