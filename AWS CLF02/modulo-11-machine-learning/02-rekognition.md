# Amazon Rekognition

## 1. Analogia do Dia a Dia
Rekognition é como um "detector de rostos e objetos" automático em fotos e vídeos, igual ao que existe em aplicativos de fotos modernos.

## 2. O que é (definição técnica oficial AWS)
Amazon Rekognition é um serviço de machine learning que permite análise de imagens e vídeos para identificar objetos, pessoas, textos, atividades e detectar conteúdo impróprio.
