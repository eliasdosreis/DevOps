# Módulo 4 — Code Review e Pull Requests
## Aula 03: Vocabulário de Código e Git em Inglês (Pronúncia Focada)

**1. SITUAÇÃO DO DIA A DIA (contexto real)**
Você está compartilhando a tela no Zoom fazendo *Pair Programming* ou destrinchando um código Terraform. Na hora de explicar o que o comando Git vai fazer ou que existe uma variável, você "engole" as palavras, pronuncia o R como H brasileiro e o gringo precisa pedir para confirmar se você disse *Repository* ou *Hepository*. A maneira verbal como lidamos com a sintaxe define a fluência técnica.

**2. O QUE VOCÊ PRECISA DIZER (objetivo da aula)**
Nesta aula, não focaremos em frases longas, mas em dominar o léxico/vocabulário principal do ciclo "Commit -> Merge -> Deploy", incluindo a pronúncia de palavras incrivelmente difíceis para brasileiros. Saber falar "Branch" ou "Variable" com a fonética do inglês é crucial.

**3. VERBOS REGULARES E ESSENCIAIS DE GIT / DEV**
* **To Push** (Empurrar / Enviar)
  * Nunca diga apenas "I pushed" (ai puxed). O som de "ed" no fim costuma ser sutil. "I push-t".
  * *Exemplo:* "I just pushed the branch to remote."
* **To Pull** (Puxar / Baixar)
  * A pronúncia do LL (Pôw). Nunca confunda pull (puxar) com push (empurrar) na call.
  * *Exemplo:* "Pull the latest changes before compiling."
* **To Commit** (Comitar/Registrar)
  * A sílaba tônica não é no 'Co' e sim no 'mit' (co-MÍT).
  * *Exemplo:* "Did you commit the changes?"
* **To Revert** (Reverter um erro)
  * Usa-se quando você apaga via controle de versão.
  * *Exemplo:* "We can just revert this commit entirely."
* **To Stash** (Guardar modificações pendentes/temporárias)
  * *Exemplo:* "Just stash your changes and switch branches."

**4. VOCABULÁRIO TÉCNICO (CUIDADO COM A PRONÚNCIA)**
| Palavra/Técnica | Pronúncia PT para Facilitar | Exemplo no DevOps |
| :--- | :--- | :--- |
| **Repository** | re-PO-zi-to-ri (O 'R' como em 'porta' caipira) | "Clone the **repository** first." |
| **Variable** | VÉ-ria-bow | "Declare an environment **variable**." |
| **Branch** | BRENTCH | "Create a new feature **branch**." |
| **To Merge** | MÊRDJ | "Let's **merge** it to master/main." |
| **Conflict** | CÓN-flict | "There is a merge **conflict** here." |

**5. MINI DIÁLOGO REAL**
*[Screen Share - Troubleshooting]*
**You:** Okay, the pipeline broke. Let me pull the latest changes from the main branch.
*(Ok, o pipeline quebrou. Deixe-me baixar as últimas mudanças da main branch.)*

**Teammate:** Did you stash your current work? You might get a merge conflict.
*(Você guardou no stash seu trabalho atual? Você pode sofrer um conflito de merge.)*

**You:** Oh, right. Let me stash them. Okay, I've pulled. Now I'll commit the S3 fix and push it to the remote repository.
*(Ah, verdade. Deixe-me guardar. Ok, já baixei. Agora vou commitar o conserto do S3 e mandar pro repositório remoto.)*

**6. ERROS COMUNS DE BRASILEIRO EM INGLÊS / PRONÚNCIA**
❌ **"I give a push / I make a commit."** (Dar ou fazer não existe).
✅ **"I pushed the changes." / "I committed the code."** (Em inglês as ações do Git viraram o próprio verbo inteiro).

❌ **Pronunciar Repo como "HE-pô"** (Com R de rato).
✅ **Pronunciar "RE-po"** (Com R caipira enrolado - igual Reto/Retro, RRREpo).

❌ **"Variable" pronunciado como "Va-ri-a-ble"**
✅ **Vér-ia-bowl.**

**7. EXERCÍCIO PRÁTICO**
Crie mentalmente as seguintes frases usando apenas verbos (To push, To commit, To pull, To merge):
1. "Eu comitei as mudanças e empurrei/pushei para o GitHub." (I committed...)
2. "Nós precisamos mesclar/mergear esse código." (We need...) 

**8. FRASE PARA MEMORIZAR HOJE**
🗣️ **"I've pushed the latest commit to my branch. Let's merge it once the checks pass."**
*(Eu enviei o último commit para a minha branch. Vamos mergear isso assim que as checagens passarem.)*

**9. DICA PARA ENTREVISTA**
❓ Jamais na entrevista hesite na palavra 'Repository', treine falar a palavra completa: *"I maintain the infrastructure repository."* O cérebro gringo capta o sotaque duro rápido, mas se o jargão do Git for fluente, o "tecnês" garante que eles confiem em você sem erro de interpretação.
