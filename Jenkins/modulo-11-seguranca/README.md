# Módulo 11 — Segurança e Governança

## O que você vai aprender

- Modelo de segurança do Jenkins: autenticação e autorização
- Role-Based Access Control (RBAC) com Matrix Authorization
- Script Security: aprovação de scripts Groovy
- Auditoria de builds e rastreabilidade
- Boas práticas de hardening do Jenkins em produção

## Arquivos deste módulo

| Arquivo | Descrição |
|---------|-----------|
| `11-Jenkinsfile-script-security.groovy` | Demonstra Script Security e aprovações |
| `11-rbac-config.md` | Guia de configuração de RBAC no Jenkins |
| `11-hardening-checklist.md` | Checklist de hardening para produção |
| `11-auditoria.md` | Configuração de auditoria e rastreabilidade |
| `11-conceitos.md` | Explicação teórica completa do módulo |

## Ordem de estudo

1. Leia `11-conceitos.md`
2. Aplique o `11-rbac-config.md` no seu Jenkins
3. Revise `11-hardening-checklist.md`
4. Execute `11-Jenkinsfile-script-security.groovy`
