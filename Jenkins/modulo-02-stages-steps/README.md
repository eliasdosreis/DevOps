# Módulo 2 — Stages, Steps e Flow Control

## O que você vai aprender

- Múltiplos stages e ordem de execução
- Steps essenciais: sh, echo, bat, script
- Bloco `post`: always, success, failure, unstable, cleanup
- Diretiva `when`: condições para executar ou pular stages
- Execução paralela de stages com `parallel`

## Arquivos deste módulo

| Arquivo | Descrição |
|---------|-----------|
| `02-Jenkinsfile-multi-stages.groovy` | Pipeline com múltiplos stages sequenciais |
| `02-Jenkinsfile-steps-essenciais.groovy` | Demonstra sh, echo, bat, script |
| `02-Jenkinsfile-post.groovy` | Uso completo do bloco post |
| `02-Jenkinsfile-when.groovy` | Condicionais com a diretiva when |
| `02-Jenkinsfile-parallel.groovy` | Execução paralela de stages |
| `02-conceitos.md` | Explicação teórica completa do módulo |

## Ordem de estudo

1. Leia `02-conceitos.md`
2. Execute na ordem: multi-stages → steps → post → when → parallel
