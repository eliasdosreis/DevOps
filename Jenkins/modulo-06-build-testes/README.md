# Módulo 6 — Build e Testes

## O que você vai aprender

- Build com Maven, Gradle, npm, pip (exemplos de cada)
- Execução e publicação de relatórios de testes (JUnit, pytest)
- Análise de qualidade de código com SonarQube
- Cobertura de código e quality gates
- Artefatos de build: arquivamento e fingerprinting

## Arquivos deste módulo

| Arquivo | Descrição |
|---------|-----------|
| `06-Jenkinsfile-maven.groovy` | Build completo com Apache Maven |
| `06-Jenkinsfile-gradle.groovy` | Build completo com Gradle |
| `06-Jenkinsfile-npm.groovy` | Build de aplicação Node.js com npm |
| `06-Jenkinsfile-python.groovy` | Build e testes de projeto Python com pip/pytest |
| `06-Jenkinsfile-sonarqube.groovy` | Análise de qualidade com SonarQube |
| `06-Jenkinsfile-artefatos.groovy` | Arquivamento de artefatos e fingerprinting |
| `06-conceitos.md` | Explicação teórica completa do módulo |

## Ordem de estudo

1. Leia `06-conceitos.md`
2. Escolha sua stack principal (Maven, npm, Python) e execute esse primeiro
3. Execute sonarqube após configurar o servidor SonarQube
4. Execute artefatos para entender o sistema de arquivamento
