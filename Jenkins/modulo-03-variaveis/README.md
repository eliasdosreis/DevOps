# Módulo 3 — Variáveis, Parâmetros e Environment

## O que você vai aprender

- Variáveis de ambiente built-in do Jenkins (BUILD_NUMBER, GIT_BRANCH, etc.)
- Bloco `environment`: definindo variáveis customizadas
- Parâmetros de pipeline: string, boolean, choice, password
- Escopo de variáveis: global vs stage vs step

## Arquivos deste módulo

| Arquivo | Descrição |
|---------|-----------|
| `03-Jenkinsfile-env-builtin.groovy` | Variáveis nativas do Jenkins |
| `03-Jenkinsfile-environment.groovy` | Bloco environment com variáveis customizadas |
| `03-Jenkinsfile-parametros.groovy` | Todos os tipos de parâmetros de entrada |
| `03-Jenkinsfile-escopo.groovy` | Demonstra escopo global, stage e step |
| `03-conceitos.md` | Explicação teórica completa do módulo |

## Ordem de estudo

1. Leia `03-conceitos.md`
2. Execute na ordem: env-builtin → environment → parametros → escopo
