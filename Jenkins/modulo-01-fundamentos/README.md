# Módulo 1 — Fundamentos de Pipeline

## O que você vai aprender

- O que é um Jenkinsfile e por que ele existe
- Pipeline Declarativo vs Scripted: diferenças e quando usar cada um
- Estrutura básica: pipeline, agent, stages, stage, steps
- Hello World pipeline e anatomia completa do build log

## Arquivos deste módulo

| Arquivo | Descrição |
|---------|-----------|
| `01-Jenkinsfile-hello-world.groovy` | Pipeline mínimo — apenas imprime mensagem |
| `01-Jenkinsfile-declarativo-vs-scripted.groovy` | Comparação lado a lado dos dois estilos |
| `01-Jenkinsfile-anatomia.groovy` | Pipeline com comentários em cada elemento |
