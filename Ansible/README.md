## 🗺️ Mapa do Projeto (Como um Mapa do Tesouro!)

Imagine que aprender Ansible é como construir uma casa:
- **Módulo 1** = Fundação e ferramentas básicas
- **Módulo 2** = Paredes e estrutura
- **Módulo 3** = Acabamento avançado
- **Módulo 4** = Construindo a casa completa (Projeto Final Senior)
