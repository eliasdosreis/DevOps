# 🚀 Curso Completo: Maven & Gradle — Do Zero ao Senior

> Aprenda Build Tools de forma profunda, incremental e com foco em entrevistas técnicas de nível Senior.

---

## 📁 Estrutura do Curso

```
Maven, Gradle/
│
├── modulo-00-preparacao/          → Instalação, ambiente e primeiros passos
├── modulo-01-maven-fundamentos/   → Ciclo de vida, pom.xml, repositórios
├── modulo-02-maven-dependencias/  → Escopos, transitivas, BOM, conflitos
├── modulo-03-maven-plugins/       → Plugins essenciais e criação de JARs
├── modulo-04-maven-multi-modulo/  → Projetos pai/filho, herança, agregação
├── modulo-05-gradle-fundamentos/  → DAG, DSLs, build.gradle, wrapper
├── modulo-06-gradle-dependencias/ → Configurações, BOM, Version Catalog
├── modulo-07-gradle-tasks/        → Tasks customizadas, cache, inputs/outputs
├── modulo-08-gradle-plugins/      → Plugins core, comunidade, buildSrc
├── modulo-09-gradle-multi-modulo/ → Subprojetos, convention plugins
├── modulo-10-performance/         → Build Cache, paralelismo, Build Scan
├── modulo-11-cicd/                → GitHub Actions, Nexus, GPG, SemVer
└── modulo-12-projeto-final/       → Projeto Senior completo + simulação entrevista
```

---

## 🎯 Objetivo

Ao concluir todos os módulos, você será capaz de:

- ✅ Criar e gerenciar projetos reais com Maven e Gradle
- ✅ Entender cada conceito profundamente (não apenas decorar comandos)
- ✅ Passar em entrevistas técnicas de nível **Senior**
- ✅ Repassar esse conhecimento para outros (nível mestria)
- ✅ Usar esse projeto como base para qualquer projeto Java/Kotlin

---

## 📚 Como Estudar

Cada módulo segue **sempre** esta estrutura:

1. **ANALOGIA DO DIA A DIA** — conceito com exemplo do mundo real
2. **O QUE É** — definição técnica precisa (nível Senior)
3. **ARQUIVO COMENTADO** — cada linha explicada
4. **COMANDOS PASSO A PASSO** — com saída esperada e troubleshooting
5. **VERIFICAÇÃO** — como confirmar que funcionou
6. **CONCEITO SENIOR** — o "porquê profundo" que Junior não sabe
7. **PERGUNTA DE ENTREVISTA** — perguntas reais + respostas esperadas
