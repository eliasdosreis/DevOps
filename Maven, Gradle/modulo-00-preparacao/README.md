# Módulo 0 — Preparação do Ambiente

## O que você vai aprender neste módulo

- Instalar JDK, Maven e Gradle corretamente
- Configurar variáveis de ambiente (JAVA_HOME, M2_HOME, PATH)
- Verificar que tudo está funcionando
- Entender a estrutura de diretórios padrão de projetos Java

## Arquivos deste módulo

| Arquivo | Conceito |
|---------|----------|
| `01-instalacao-e-variaveis.md` | Como instalar JDK, Maven e Gradle + variáveis de ambiente |
| `02-estrutura-de-diretorios.md` | Estrutura padrão de diretórios (Maven Standard Directory Layout) |
| `03-primeiros-comandos.md` | Primeiros comandos para verificar instalação |
| `HelloWorld.java` | Arquivo Java simples para testar o ambiente |
