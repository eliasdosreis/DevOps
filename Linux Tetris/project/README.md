Linux Tetris CLI - Educacional

Como rodar:
1. Abra `project/index.html` em um navegador moderno.
2. Ou rode um servidor local, por exemplo:

```bash
cd "c:\Users\Elias\Documents\Estudo\Linux Tetris\project"
python -m http.server 8000
# então abra http://localhost:8000
```

Descrição:
- Jogo inspirado em Tetris que faz perguntas sobre comandos Linux antes de cada peça.
- Responder corretamente dá bônus, errar adiciona lixo e acelera o jogo.

Arquivos:
- index.html, style.css, engine.js, game.js, questions.js
