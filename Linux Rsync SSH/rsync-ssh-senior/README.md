# 🚀 Projeto Senior: Migração com rsync + SSH
## Alex DevOps Coach | Trilha Senior

---

## 📚 O que você vai aprender?

Imagine que você precisa **mover uma caixa cheia de livros** de uma casa para outra.
- O **rsync** é o caminhão que leva os livros
- O **SSH** é a chave segura que abre a porta da outra casa
- Juntos, eles fazem migrações de arquivos de forma **rápida, segura e profissional**
